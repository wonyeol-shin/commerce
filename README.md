# 커머스 과제 도전 step 1 완료

---
### Product : 상품이름, 가격, 설명, 재고 필드를 가짐
### Category : 상품의 카테고리에 따라 상품(Product) 리스트를 가지고 있음 (상품의 분리)
### Main : 주문 프로그램을 실행
### CommerceSystem : 카테고리 추가, 상품추가, 카테고리 별 상품List를 가지고 있음, 카테고리 별 상품을 출력해주고 고객이 추가한 장바구니의 상품 주문을 처리
### Cart : 고객이 장바구니에 추가한 상품와 수량을 가짐
### Customer : 고객의 이름, 이메일, 등급을 가짐, 추후 step3에서 고객의 등급에 따라 할인율을 다르게 할 수 있음

---

Main의 commerceSystem.startProgram(); 를 통해 프로그램 시작 가능
